# 🎯 Personal Introduction Program

A simple Python program that asks the user for their name, age, hobby, and city, then displays a friendly, personalized welcome message.

## 📖 About

This is a beginner-friendly Python project built as part of a 5-day learning challenge covering:
- Setting up Python and running a first program
- Using `input()` to collect user data
- Storing data in variables
- Using f-strings to format and display output
- Adding extra questions and polishing the final output

## 🛠️ Requirements

- Python 3.x

No external libraries are required — this project only uses Python's built-in features (see `requirements.txt`).

## ▶️ How to Run

1. Make sure Python 3 is installed on your system. Check with:
   ```bash
   python3 --version
   ```
2. Clone this repository or download `personal_intro.py`.
3. Run the program:
   ```bash
   python3 personal_intro.py
   ```
4. Answer the prompts when asked.

## 📊 Sample Output

```
What is your name? Alex
```
